Source types

MARqeT uses different source types, ensuring all customers can link to their own inventories to select observation units:

Source type Description
Nursery Field Books Observation units are defined in the PRISM database.
QA Nautilus Observation units are already prepared and defined in a QA Nautilus worklist.
External sources Observation units originate from outside the BASF environment.

MARqeT also provides the option to re-analyze samples:

Source type Description
DNA plates Re-analyze entire plates that were already extracted.
DNA cherry pick Re-analyze specific samples from plates that were already extracted.
