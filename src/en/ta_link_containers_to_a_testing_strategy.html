Link containers to a testing strategy

A customer made a request, generated a sample container and provided input for the testing strategy. A test strategist finalized the testing strategy.
Note: Both a test strategist and a customer can finalize the linkage matrix.

For large requests, your testing strategy might only be applicable to a subset of sample containers. For these requests, you can link each sample container to the appropriate strategy by defining a matrix. This matrix contains the different testing strategies and different sample container numbers, which you link with a simple check mark.

Important: Be aware that if you mark the Sample Container Generator or Testing Strategy module as incomplete, the linkage matrix will reset.
  1. To open the Requests list, go to the left blue menu and click Requests.
  2. Select the request you want to link sample containers for.
    The request appears in the left blue menu.
  3. In the left blue menu, under the request, select Testing strategy.
  4. At the right top of the screen, click Link containers.
    Tip: Alternatively, click next to a service type to link containers to that specific service type.
  5. At the top of the screen, apply a Project type, Analysis type and Testing strategy filter to narrow down the testing strategies.
  6. Under filters, slide the container range bullets left and right to narrow down the displayed containers.
  7. Optionally, unfold the testing strategies to component level in the left column of the table.
  8. Click in the correct cells to add a check mark, indicating a linkage between the testing strategy and sample container.
    Tip: To select multiple cells at once, use the following methods:
    Purpose Method
    Select first cell, last cell and all cells in between Hold the shift key while selecting multiple cells
    Select a complete row Click on a row name
    Select a complete column Click on a column name
    Tip: Alternatively, turn on the Select all button to check multiple cells at once. Hover over the cells to see which will be checked.
  9. To finalize the linkage, go to the right top of the screen and click Mark complete.
After defining the linkage matrix, you can follow your container to the lab.
