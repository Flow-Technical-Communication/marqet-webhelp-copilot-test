Define the sample container layout

When a lab wants to perform an analysis on your sample container, they need to know the exact content of each well. To define the content of each well, that is, the layout of a sample container, you first need to define a source type. Next, MARqeT allows you to setup the screen in advance or in real-time:

  1. You can add observation units to the virtual sample container in advance and subsequently sample your containers.
  2. You can sample your containers in real-time by scanning the individual plants one by one and create the virtual sample container as you go.
Parent topic: Sample Container Generator

Using PRISM as a source

When using PRISM as a source, perform the following steps to setup the container and define its layout:

  1. Select a book.
  2. Select the observation units.
  3. Optionally, change the observation unit order.
  4. Define the container layout.

Select a book

To set up the layout of a container, first create a request.
The MARqeT sample container generator module is linked to the PRISM software. To add observation units from Nursery Field Books to the module, add these in the PRISM software. When setting up the container layout, you can retrieve observation unit information from these books.
  1. To open the Requests list, go to the left blue menu and click Requests.
  2. Select the request you want to generate sample containers for in the Request list.
    The request appears in the left blue menu.
  3. In the left blue menu, under the request, select Sample container generator.
  4. Select PRISM nursery field book as a source type.
  5. Select how you want to add your field books: Manual or By uploading a file.
    Note: To ensure you use the correct format, download the available template. If you choose by file (with positions), add the correct positions to each row.
  6. If you selected Manual, add the books you want to use to fill the sample container:
    1. Click Add.
      The tissue type you selected when creating a request will define the books you can select: for plants nursery field books are available, while inventory books are used for seeds.
    2. Check the correct books in the pop-up window and click Add.
      Tip: Use the filters at the top of the list to search specific books.
    3. Optionally, define the sample container options. Remember, fixed empties are used for control, while floating empties identify the container.
  7. Click Next.
If you selected Manual, the next step is to select the observation units. If you selected By file, you can immediately continue to defining the container layout contents.

Select the observation units

First select a book.
After filling in all information for your sample container setup, you select the correct observation units you want to add to your container.
  1. If you selected more than one book, select the correct book at the top of the list.
    Tip: If necessary, click Setup in the trail navigator on top to go back. Click Clear next steps to edit setup to change the books.
  2. Select the correct observation units in the list.
    Tip: Use the filters at the top of the list to search for specific observation units. Hold the shift key to select two observation units and all observation units in between.
  3. Click > Mark (#) for sampling.
    Note: To correct an error, select the incorrect observation units and click > Unmark (#) for sampling.
  4. Click Next.
    Important: Once you add an observation unit from a Nursery Field book, an ID is generated by MARqeT in the background. The ID captures the link to the original observation unit in PRISM to ensure traceability.
The next step is to define the container layout or optionally change the observation unit order first.

Change the observation unit order

First select a book and select the observation units.
For large requests with many sample containers, you can sort observation units (to a degree) without having to review each well. Only observation units selected during the observation unit selection are available in this list. Observation units are displayed in the default order of the book.
Important: This feature is not created to micromanage each setup of the sample container. For highly customized cases, use the container layout screen.
  1. Select the observation units you want to re-order.
    Tip: If necessary, click Setup or Observation unit select in the trail navigator on top to change the books or observation units.
  2. Click and select the way you want to re-order:
    Option Outcome
    Order Order the selection based on specific columns:
    1. Select the fields you want to use to order ascending or descending.
    2. Click Order.
    Move to beginning Move the selection to the beginning of the list.
    Move to position Move the selection to a specific position:
    1. Enter a list position number.
    2. Click Move.
    Move to end Move the selection to the end of the list.
    Remove Remove the selection.
  3. Click Next.
The next step is to define the container layout.

Define the container layout

To set up the layout of a sample container, first select the field books to sample from, select the observation units, and optionally change the observation unit order. Additionally, check the criteria for the container layout and the filling options.
After changing the observation unit order, you define the layout of your sample container.
  1. If you want to assign all observation units to the sample container, click > Assign all observation units by order #.
    Notice: All previous defined sample container layouts are cleared when selecting this option.
  2. If you want to assign specific observation units to the sample container, select the observation units, click and choose a filling option:
    Option Outcome
    Add selected observation units (#) to selected position Add the selected observation units to the selected well.
    Assign selected observation units (#) to empty positions as from selected position Add the selected observation units to the first empty wells starting from the selected well.
    Assign selected observation units (#) after last filled position Add the selected observation units to the first empty wells after the last filled well.
    Note: The following tips may help you fill a sample container:
    • Hold the shift key to select two observation units and all observation units in between.
    • For small setups, select next to the observation unit of your choice and drag to the correct well on the sample container.
    • If no selection is made, all observation units are implicitly selected.
    • To empty a well, go to the list view of the sample container and select > Clear positions.
  3. To finalize the sample container setup, go to the top right of the screen and click Mark complete.
    Important: To regain the ability to make adjustments, click Mark incomplete. Be aware that this will reset the linkage matrix.
In the next step, you need to create a testing strategy.

Using QA Nautilus worklists

To set up the layout of a sample container, first create a request.
If you use Nautilus as your observation units source, prepare your observation units, their order and the sample container setup as a worklist in the Nautilus tool. When setting up the container layout, MARqeT automatically configures the sample containers.
  1. To open the Requests list, go to the left blue menu and click Requests.
  2. Select the request you want to generate sample containers for in the Request list.
    The request appears in the left blue menu.
  3. In the left blue menu, under the request, select Sample container generator.
  4. Select QA Nautilus as a source type.
  5. Select the container type.
    Important: Be aware that the worklist must be ready in Nautilus to synchronize with MARqeT.
  6. Select the worklist and click Next.
    MARqeT automatically generates the sample container layout.
  7. To finalize the sample container setup, go to the top right of the screen and click Mark complete.
    Important: To regain the ability to make adjustments, click Mark incomplete. Be aware that this will reset the linkage matrix.
In the next step, you need to create a testing strategy.

Using external samples

When using external samples, perform the following steps to setup the container and define its layout:

  1. Upload a csv template.
  2. Define the container layout.

Upload a csv template

To set up the layout of a sample container, first create a request.
If you use external samples as your observation unit source, MARqeT does not have a source system to use. When setting up the sample container layout, fill out the available template to provide any information necessary.
  1. To open the Requests list, go to the left blue menu and click Requests.
  2. Select the request you want to generate sample containers for in the Request list.
    The request appears in the left blue menu.
  3. In the left blue menu, under the request, select Sample container generator.
  4. Select External as source type.
  5. Download and fill in the csv template.
    Note: The csv template has the following fields:
    Observation unit ID Observation unit name Source reference ID
    BASF sample ID, for example 21mrgm00125 (Mandatory) Readable name Unique ID outside BASF for traceability
    Be aware that the order of your template is the order in which the observation unit will placed in your plate.
  6. Upload the template.
  7. Select the container type, the number of fixed empties and whether or not you want to include floating empties.
  8. Click Next.
    The information from the template appears in the sample container layout screen.
The next step is to define the container layout.

Define the container layout

To set up the layout of a sample container, first upload a csv template. Additionally, check the criteria for the container layout and the filling options.
  1. If you want to assign all observation unit to the container, click > Assign all observation units by order #.
    Notice: All previous defined sample container layouts are cleared when selecting this option.
  2. If you want to assign specific observation units to the sample container, select the observation units, click and choose a filling option:
    Option Outcome
    Add selected observation units (#) to selected position Add the selected observation units to the selected well.
    Assign selected observation units (#) to empty positions as from selected position Add the selected observation units to the first empty wells starting from the selected well.
    Assign selected observation units (#) after last filled position Add the selected observation units to the first empty wells after the last filled well.
    Note: The following tips may help you fill a sample container:
    • Hold the shift key to select two observation units and all observation units in between.
    • For small setups, select next to the observation unit of your choice and drag to the correct well on the sample container.
    • If no selection is made, all observation units are implicitly selected.
    • To empty a well, go to the list view of the sample container and select > Clear positions.
  3. To finalize the sample container setup, go to the top right of the screen and click Mark complete.
    Important: To regain the ability to make adjustments, click Mark incomplete. Be aware that this will reset the linkage matrix.

Using DNA plates

To set up the layout of a sample container, first create a request.
You can re-analyze any plate that has already been extracted. Notice that you cannot sample DNA limitlessly. Always contact the lab to ensure DNA availability.
  1. To open the Requests list, go to the left blue menu and click Requests.
  2. Select the request you want to generate sample containers for in the Request list.
    The request appears in the left blue menu.
  3. In the left blue menu, under the request, select Sample container generator.
  4. Select DNA plate as source type.
  5. Select one or multiple plates from previous requests:
    • Click Add to manually search the plates
    • Download, fill in and upload a csv template
    Notice: If you select DNA plates, they will hold their original name, layout and samples.
  6. Check the sample container layout.
    Tip: See container setup screens to change the sample container layout.
  7. To finalize the sample container setup, go to the upper right corner of the screen and click Mark complete.
    Note: Be aware that you can only select plates from submitted requests.
    Important: To regain the ability to make adjustments, click Mark incomplete. Be aware that this will reset the linkage matrix.
In the next step, you need to create a testing strategy.
